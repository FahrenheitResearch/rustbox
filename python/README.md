# Python Compatibility

These directories are reserved for thin compatibility layers that sit on top of the Rust core.

- `python/metrust`: compatibility-facing Python package surface.
- `python/compat-cfgrib`: cfgrib-style adapters backed by Rust GRIB decode.
- `python/compat-eccodes`: ecCodes-style adapters backed by Rust GRIB decode.
- `python/compat-herbie`: Herbie-style fetch helpers backed by Rust fetch planning.
- `python/compat-wrf`: WRF-style adapters backed by Rust diagnostics.
